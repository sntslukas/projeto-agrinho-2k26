# projeto-agrinho-2k26
# Monocultura: Vantagens Econômicas e Influência no Crescimento do PIB Brasileiro

## 🎯 Objetivo da Página
O objetivo desta página é fornecer uma experiência digital conceitual, contemporânea e altamente profissional sobre o impacto macroeconômico dos sistemas de monocultura no Brasil (soja, milho, cana-de-açúcar, café e algodão). A interface foi projetada para combinar a robustez do agronegócio tradicional com a sofisticação da inovação tecnológica digital e da inteligência artificial, servindo como uma central informativa dinâmica para estudantes, analistas e entusiastas do setor logístico e financeiro nacional.

## 🛠️ Tecnologias Utilizadas
- **HTML5 Semântico:** Estruturação robusta e nativamente inteligível para mecanismos de busca e tecnologias assistivas.
- **CSS3 Avançado (Modern Layout Engine):** Utilização de CSS Grid, Flexbox, e funções dinâmicas matemáticas como `clamp()` para alcançar total fluidez responsiva de 320px até 2560px (Ultra-wide monitors) sem dependência de frameworks.
- **JavaScript Moderno (ES6+):** Código limpo, desacoplado, sem dependência de bibliotecas externas, operando via escuta de eventos nativos (`addEventListener`).

## ♿ Recursos de Acessibilidade Embutidos
No canto superior direito da tela, há uma central flutuante inteligente equipada com:
1. **Aumento e Diminuição Gradual da Fonte:** Ajustes na escala tipográfica sem quebrar os contêineres e o grid do layout.
2. **Alternância de Tema (Modo Escuro / Claro):** Inversão inteligente de contraste para leitura confortável em ambientes de baixa luminosidade.
3. **Leitura de Voz por Síntese Nativa:** Integração avançada com a *Web SpeechSynthesis API*, programada para ler exclusivamente o fluxo textual principal, ignorando elementos de interface e códigos do formulário.

## 🚀 Instruções de Uso

1. **Navegação Local:**
   - Faça o download dos três arquivos fundamentais (`index.html`, `style.css`, `script.js`) no mesmo diretório/pasta de sua escolha.
   - Certifique-se de manter os nomes dos arquivos exatamente como especificado.
   - Dê um duplo clique ou abra o arquivo `index.html` através de qualquer navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari).

2. **Injeção de Imagens Reais (Customização):**
   - O layout possui 3 placeholders reservados estritamente sob as especificações solicitadas (`<div class="imagem"> <img src="Foto1.png" ...></div>`).
   - Para exibir as suas imagens personalizadas, basta salvar as mídias desejadas no formato `.png` na mesma pasta dos arquivos de código, nomeando-as respectivamente como: `Foto1.png`, `Foto2.png`, e `Foto3.png`.

3. **Interação com Recursos:**
   - Interaja com os cards clicando ou focando neles através do teclado para visualizar os comportamentos e transformações do efeito hover suave.
   - Preencha os campos estruturados da barra lateral (Nome, E-mail, Cidade, Estado, País) e teste o validador nativo de e-mails do formulário clicando em "Inscrever-se no Relatório".
